# Dailymotion Downloader

Application Android simple pour télécharger des vidéos Dailymotion.

## Téléchargement

[![Télécharger APK](https://img.shields.io/badge/Télécharger-APK-green?style=for-the-badge&logo=android)](https://github.com/Vaal34/Dailymotion-Downloader-Android-APK/releases/latest/download/dailymotion-downloader.apk)

## Fonctionnalités

- Interface simple style Windows 95
- Téléchargement de vidéos Dailymotion en MP4
- Partage depuis l'app Dailymotion
- Historique des téléchargements
- Notifications de progression
- Compatible Android 5.0+

## Utilisation

1. Copiez le lien d'une vidéo Dailymotion
2. Collez-le dans l'application
3. Appuyez sur "Télécharger"

Ou partagez directement depuis l'app Dailymotion vers cette application.

## Installation

1. Téléchargez `dailymotion-downloader.apk`
2. Ouvrez le fichier sur votre téléphone
3. Autorisez l'installation depuis des sources inconnues
4. Installez et profitez!
